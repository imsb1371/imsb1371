CollabSense addresses a core bottleneck in scientific research: the overwhelming volume of literature new researchers must navigate. By combining transformer-based semantic embeddings with a collaborative web interface, the system enables teams to co-annotate, cluster, and identify gaps in a body of papers in a fraction of the time.

**Key contributions:**
- Novel UI for simultaneous multi-user annotation with conflict resolution
- Automated semantic clustering using sentence-transformers + UMAP
- Gap detection via citation graph analysis
- Published at CHI 2024 (Best Paper Nominee)
